# Student Growth - Alumni Mentorship System (WebApp)

A comprehensive web application for tracking student growth and facilitating alumni mentorship at Government College of Engineering, Erode.

## Project Overview

This is a multi-phase project to build a complete student growth tracking and alumni mentorship system. The webapp will include 8 phases of development.

### Current Phase: Phase 1
- Landing page with institution information
- Statistics display (students, departments, batches)
- Navigation structure
- Responsive design

## Getting Started

### Prerequisites
- Modern web browser
- (Optional) Node.js for local development server

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/shahana1418/student-growth-webapp.git
cd student-growth-webapp
```

2. Open in browser:
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     npx http-server
     ```

3. Visit: `http://localhost:8000`

## Development Phases

1. **Phase 1** (Current) - Landing Page & Basic Structure
   - Homepage with institution info
   - Statistics display
   - Basic navigation

2. **Phase 2** - Authentication & Login
   - User login system
   - Role-based access

3. **Phase 3** - Department Browsing
   - Department listings

4. **Phase 4** - Student Profiles
   - Student information management
   - Profile pages

5. **Phase 5** - Mentorship Matching
   - Alumni matching algorithm
   - Mentorship pairing

6. **Phase 6** - Progress Tracking
   - Growth metrics
   - Achievement tracking

7. **Phase 7** - Analytics & Reporting
   - Dashboards
   - Reports generation

8. **Phase 8** - Admin Panel & Advanced Features
   - Admin management interface
   - System administration
